The Appointment Scheduling Platform

The purpose of this application is to allow for the management of a companies appointments with their customers.

Author: Brian Rodenborn
Contact: broden3@wgu.edu
Application Version: 14.3.1
Date: October 26, 2023

IDE: Intellij Community 2023.1
JDK: Java 17.0.6
SDK: 17.0.2
MySQL Connector: mysql-connector-j-8.1.0

DIRECTIONS:
    1: Launch the Application
    2: Sign in with either "Admin, Admin" or "Test, Test"
    3: Navigate between the screens by clicking on the appropriate buttons
    4: To exit the application, just click exit on the main screen.

Description of additional report: The additional report is a count of all customers displayed by country.
